1. CSS - DISPLAY
-------------

    display : inline | block | inline-block | flex

        1. block:           Occupies full width, starts on a new line (e.g., div, p, h1).
        2. inline:          Occupies only content width, stays on the same line (e.g., span, a, img).
        3. inline-block:    Combination of block and inline (e.g., button, input).
        4. none:            Hides the element.
        5. flex:            Enables flexible box layout (responsive design).
