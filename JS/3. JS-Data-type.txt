3.  DATA TYPES
-----------------

    3.1 Primitives Data types
    ------------------------------------------------------------------------------------------------------
        Types                       Example
    ------------------------------------------------------------------------------------------------------- 
    

    1. String               -       var name = "Saravanan" /  '' or  " "
    2. Number               -       var a = 10  // 10.5 // 5.5 
    3. Boolean              -       var a = true // false
    4. Undefined            -       var a  
                            -        console.log(a)  // output Undefined
    5. Null                 -       var a = null
    6. Symbol               -       var c = ( a == b ) //  output  true / False
   
    



    3.2 Non Primitive Data Types
   ------------------------------------------------------------------------------------------------------
        Types                       Example
    -------------------------------------------------------------------------------------------------------  

        1. Array  []           -      var fruits = ['Apple', 'Orane','Grapes'] 

                                           0 -> Apple
                                           1 -> Orange
                                           2 -> Grapes

                                                name :  'Saravanan',

                                            }
