            ...

    ----------

üîß What is ALTER TABLE?
 ------------------------
    ALTER TABLE is used to change the structure of an existing table in a database.

    You can use it to:

        ‚úÖ Add a new column
        ‚úÖ Modify a column
        ‚úÖ Rename a column or table
        ‚úÖ Delete (drop) a column
        ‚úÖ Add or drop constraints (like primary key, foreign key)

     -------------------------------------------------------------
    | Task              | SQL Command Example                     |
    | ----------------- | --------------------------------------- |
    | Add a column      | `ADD column_name data_type;`            |
    | Modify a column   | `MODIFY column_name new_data_type;`     |
    | Rename a column   | `RENAME COLUMN old_name TO new_name;`   |
    | Drop a column     | `DROP COLUMN column_name;`              |
    | Add a primary key | `ADD PRIMARY KEY (column_name);`        |
    | Add a foreign key | `ADD FOREIGN KEY (...) REFERENCES ...;` |
    | Rename the table  | `RENAME TABLE old_name TO new_name;`    |
    ---------------------------------------------------------------


    Example:-
    --------

        1Ô∏è‚É£ Add a Column

3Ô∏è‚É£ ‚ùå What is DROP TABLE?
--------------------------

    The DROP TABLE command is used to permanently delete a table from a database, along with all the data stored in it.

    Syntax:-
    -------
        DROP TABLE table_name;                /* Delete signle table */
        DROP TABLE table1, table2, table3;    /* To delete multiple table */


    
    Example:-
    ---------

        DROP TABLE Students;
        DROP TABLE Students, Marks;
