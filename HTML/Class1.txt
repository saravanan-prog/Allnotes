List Tag
--------
Example pattern

    ul = unOrder List (<ul> <li>...</li> </ul>)  
    -----------------
         * Apple
         * Orange
         * Mango

         - Apple
         - Mango
         - Orange

    Ol = Order List (<ol> <li>...</li> </ol>)
    ---------------
        1. Apple
        2. Mango
        3. Orange

        A. Apple
        B. Orange
        C. Mango

   Code
   ----  
       <ul>
         <li> Apple </li>
         <li> Orange </li>
         <li> Mango </li>
       </ul>
