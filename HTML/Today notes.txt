HTML => Hyper Text markup language it is helps to creating web page
     => this one is called Scripting Language.
     

DOM  => Document object model it is reprsent to blueprint how to construct your webpage.
     => this is under comes to tree level architecture.


Tags  => it is helps to construct in your webpage.

      => We represent HTML elements with angular brackets(<>) called HTML Tags.
      => this is two types

              - parired tags. [ with closing tags ]
                  For example : <p>..</p>, <h1>..</h1>, <div>...</div>

              - Non paired tags [ without closing tags ]
                   For example :  <img src='' /> 

Elements => Element  is represent tag + content
              For example : <p> Hello world </p>
            


Types of Elements => There are 4 type of Elements is there.
                
                1. Block Level Elements - [ occupied Entired row ]
                2. Inline - Elements    - [ occupied specified space ]
                3. void Elements        - [ Without closing Elements ]
                4. Semmantic Elements.  - [ machine and humman understanding elements.]


Attribute  => An Attributes is an additonal information of the tags.
           => there are two types of attribues
                1. Common Attributes -  [id,class,name,style]
                2. Special Attributes - [src, href, type ]

Attributes.
-----------
  An attribute is an additonal information of the tags.
     1. Common Attributes. - its available all html tags 

         Id     - id is unique [#]
         Class  - class is not unique [.]
         name   - name is not unique
         style  - it is defines to write inline style.
         height - increase size of elements horizontally.
         width  - increase size of elements verticaly.
         title  - it is helps to hover the text hint.
      
    2. Special Attibutes. [ Special type of information an elements.]

        href = hyperlink reference.
        src  = source.
        alt  = alternative 
        type = typical 
        action = actions or events.
        value  = default values.
        rowspan
        colspan
        border
