Array
------
  => an array is a collection of similar data types
  => it is a fixed size data structure
  => it can store multiple values of the same type

  how to declare an array in Java:
  --------------------------------
    => syntax: 
    ----------
        data_type[] array_name = new data_type[5];
        data_type array_name[] = new data_type[5];

    Example
    -------
          int number[]; // Declaration only, no memory allocated yet
          int number[] = new int[5]; // Declaration and memory allocation


        int number[] = {10,20,30,40,50}; // Declaration and initialization

        for(int i = 0; i < number.length; i++) {
            System.out.println(number[i]); // Accessing elements of the array
        }

        int secondNumber[] = new int[5]; // Declaration and memory allocation
